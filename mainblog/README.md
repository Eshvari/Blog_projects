# Blog Project

A Django-based Blog Application where users can create posts, like posts, and comment.

## Features
- User Registration
- User Login
- Create Blog Posts
- Like and Comment on Posts
- Profile Management

## Tech Stack
- Python
- Django
- SQLite
- HTML / CSS

## Installation

1. Clone the repository

git clone https://github.com/Eshvari/Blog_projects.git

2. Go to project directory

cd Blog_projects

3. Create virtual environment

python -m venv venv

4. Activate environment

venv\Scripts\activate

5. Install dependencies

pip install -r requirements.txt

6. Run server

python manage.py runserver
